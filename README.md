# Learning Management System

This is a Learning Management System (LMS) built using React for the frontend and Spring Boot for the backend. The LMS allows users to manage courses, enrollments, and other related functionalities.

## Features

- User authentication and authorization
- Course management (create, update, delete courses)
- Enrollment management (enroll in courses, view enrolled courses)
- Responsive design for both desktop and mobile devices
## Technologies Used
- React
